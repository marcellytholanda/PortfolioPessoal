Análise de Dados Climáticos: Visualização Interativa com Inteligência Artificial

Desenvolvi um **dashboard interativo** como parte do projeto de **Atividade Extensionista II** para o curso de Ciência de Dados. O objetivo foi criar uma plataforma que utilizasse **dados climáticos** em tempo real para promover a conscientização ambiental entre alunos do Ensino Médio.

Tecnologias utilizadas: Power BI, APIs externas (OpenWeatherMap, NASA), Python, SQL

O projeto incluiu a implementação de modelos de **Inteligência Artificial** para **previsões climáticas**, além de uma interface interativa que permite a visualização clara e dinâmica dos dados. Foi uma experiência rica em aprendizado sobre IA, integração de APIs e visualização de dados.
